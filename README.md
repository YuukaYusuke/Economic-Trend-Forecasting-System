# 🌍 Economic Trend Forecasting System

> 🚀 Aplikasi berbasis **Streamlit + LSTM** untuk analisis dan prediksi tren nilai tukar mata uang global secara interaktif.

---

## ✨ Preview

💡 Dashboard modern dengan:
- 📊 Visualisasi tren
- 🤖 Prediksi AI (LSTM)
- 🌐 Data kurs realtime
- 🌏 Multi-language (termasuk aksara Jawa 😏)

---

## ⚙️ Tech Stack

- 🐍 Python 3.10 – 3.11
- 📈 TensorFlow / Keras (LSTM)
- 🎨 Streamlit
- 🌐 Alpha Vantage API / Frankfurter API
- 📊 Plotly / Matplotlib

---

## 🧩 Fitur Utama

- 📊 **Dashboard Eksekutif**
- 🤖 **Prediksi LSTM + Confidence**
- 🌐 **Realtime Currency Data**
- 📈 **Visualisasi Interaktif**
- 🌏 **Multi-language (13 bahasa + Jawa)**
- 📤 **Export CSV / PDF**
- 🔍 **Perbandingan & Simulasi Mata Uang**
- 🌙 **Dark Mode UI**
